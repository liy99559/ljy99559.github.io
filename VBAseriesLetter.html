Generate a series of letters from Excel

Overview

This project aims to generate a large amount of letters based on a tamplate .docx document and insert specific text and images according to the information stored in Excel.

Documents in hand

Firstly, we need to prepare a desired template document in .docx format. For example: 

Date: The date 
Dear: Recipient’s name

Content of letter

We want to insert some specific text according to recipient’s property here: 

We want to insert a specific image according to recipient’s property into the next line
Image


Best regards,
XXX

Then we also need an Excel worksheet containing the information of all recipients, named Data.xlsx. For example:

Name Date Number Condition
Alpha 01/02/2016 3000 1
Beta 08/11/2016 4000 2
Gamma 09/04/2016 5000 3
Theta 28/08/2016 6000 2

VBA code

Open the template.docx document, great Macro, the code is as following:

Sub Generator()

     ' Exit

      Exit Sub  
    End If
   ' Read Data.xlsx file  
    Dim ExcelObject As Object  
    Set ExcelObject = GetObject(FilePath)  
     Set Table = ExcelObject.Sheets(1).UsedRange()

      Dim imagePath As String

         For Each Var In ActiveDocument.Variables  
         Var.Delete  
       Next

        ActiveDocument.Variables.Add Name:="Name", Value:=Table.Cells(i, 1).Text
     ' Variable: Number£¨The value is in Data.xlsx row i column 3  
        ActiveDocument.Variables.Add Name:="Number", Value:=Table.Cells(i, 3).Text
     ' Variable: Date£¨The value is in Data.xlsx row i column 2  
        ActiveDocument.Variables.Add Name:="Date", Value:=Table.Cells(i, 2).Text
     'add text based on conditions  
         condition = Table.Cells(i, 4).Text  
         Select Case condition  
          Case Is = "1"  
            ActiveDocument.Variables.Add Name:="condition", Value:="Text " & "for  condition 1"  
          Case Is = "2"  
            ActiveDocument.Variables.Add Name:="condition", Value:="Text for condition 2"

         End Select

          If i <> 2 Then

             Selection.Delete  
          End If

      ' save the file, file name£∫Name.docx  
          ActiveDocument.SaveAs2 filename:=Table.Cells(i, 1).Text & "_" & Table.Cells(i, 3).Text & "_Offer_letter" & ".docx", FileFormat:=wdFormatXMLDocument, LockComments:=False, Password:="", AddToRecentFiles:=True, WritePassword:="", ReadOnlyRecommended:=False, EmbedTrueTypeFonts:=False, SaveNativePictureFormat:=False, SaveFormsData:=False, SaveAsAOCELetter:=False, CompatibilityMode:=15

    Next

End Sub

Before we run the above Macro, we need to create doc variable in the template document at the position where we want to insert data from the Excel worksheet.
Put the curser at the desired position and press Ctrl+F9, it should appear like this

Date: { }
Dear: { }

Then type in the curly brakets with { DOCVARIABLE Name }or any variable name we need.

Date: { DOCVARIABLE Date }
Dear: { DOCVARIABLE Name }

In order to determine the position of inserting the images, we need a bookmark. Go to the position requires the image, click insert button then select bookmark, name the bookmark as "photobm", or whatever other names.

Note: the image name should be the same as the corresponding conditions in Excel
